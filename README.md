# Αιτήσεις Αδειών Δικαστικής Αστυνομίας

Διαδικτυακή εφαρμογή για την εύκολη συμπλήρωση και έκδοση αιτήσεων αδειών για το προσωπικό της Δικαστικής Αστυνομίας.

🔗 **[Άνοιγμα Εφαρμογής](https://ale161.github.io/dda-adeies/)**

## Δυνατότητες

*   **Αυτόματη Συμπλήρωση:** Επιλέξτε την Εισαγγελία σας και τα στοιχεία (Διεύθυνση, Τηλέφωνο, Πόλη, Προϊστάμενος) συμπληρώνονται αυτόματα.
*   **Πλήρης Λίστα Αδειών:** Περιλαμβάνει όλους τους τύπους αδειών (Ομάδα Α & Β) σύμφωνα με την τελευταία εγκύκλιο.
*   **Ζωντανή Προεπισκόπηση:** Βλέπετε το έγγραφο να δημιουργείται καθώς πληκτρολογείτε.
*   **Εξαγωγή PDF:** Κατεβάστε έτοιμο προς εκτύπωση αρχείο PDF, απόλυτα συμβατό με τα υπηρεσιακά πρότυπα.
*   **Λειτουργία Offline (PWA):** Η εφαρμογή λειτουργεί ακόμα και χωρίς internet!

## Εγκατάσταση ως Εφαρμογή (PWA)

Μπορείτε να εγκαταστήσετε την εφαρμογή στο κινητό ή τον υπολογιστή σας για γρήγορη πρόσβαση και λειτουργία χωρίς σύνδεση.

### Σε Android (Chrome)
1.  Ανοίξτε το [link της εφαρμογής](https://ale161.github.io/dda-adeies/).
2.  Πατήστε τις 3 τελείες (μενού) πάνω δεξιά.
3.  Επιλέξτε **"Εγκατάσταση εφαρμογής"** ή **"Προσθήκη στην αρχική οθόνη"**.

### Σε iPhone/iPad (Safari)
1.  Ανοίξτε το [link της εφαρμογής](https://ale161.github.io/dda-adeies/) στο Safari.
2.  Πατήστε το κουμπί **Κοινοποίηση (Share)** (το τετράγωνο με το βέλος προς τα πάνω).
3.  Επιλέξτε **"Προσθήκη στην Οθόνη Αφετηρίας" (Add to Home Screen)**.

### Σε Υπολογιστή (Chrome/Edge)
1.  Ανοίξτε το [link της εφαρμογής](https://ale161.github.io/dda-adeies/).
2.  Πατήστε το εικονίδιο εγκατάστασης (οθόνη με βέλος) που εμφανίζεται δεξιά στη γραμμή διευθύνσεων.
3.  Πατήστε **"Εγκατάσταση"**.

## Τεχνικές Πληροφορίες

Η εφαρμογή είναι κατασκευασμένη με σύγχρονες τεχνολογίες web:
*   **React 19** & **TypeScript**
*   **Tailwind CSS 4** για το styling
*   **jsPDF** για την παραγωγή εγγράφων στον browser
*   **Vite PWA** για την offline λειτουργία

## Ανάπτυξη (Development)

Για να τρέξετε την εφαρμογή τοπικά:

1.  Κλωνοποιήστε το repository:
    ```bash
    git clone https://github.com/ale161/dda-adeies.git
    ```
2.  Εγκαταστήστε τις εξαρτήσεις:
    ```bash
    pnpm install
    ```
3.  Ξεκινήστε τον server ανάπτυξης:
    ```bash
    pnpm dev
    ```
